Русский перевод для трейнера Rampage, который используется для игры RDR2.

Инструкция: Скачать файл Russian.xml и переместить по примеру пути в трейнер: C:\Games\Red Dead Redemption 2\RampageFiles\Language

Оригинальный английский трейнер можно скачать отсюда: https://www.nexusmods.com/reddeadredemption2/mods/233

Над русификатором работают: Canek77 (вы также можете присоединиться)

Для поддержки донатом: https://www.donationalerts.ru/r/canek77
